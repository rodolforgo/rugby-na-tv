# Rugby na TV

Plataforma para compartilhamento e divulgação de transmissões de rugby no Brasil, centralizando informações sobre jogos, competições e streamings de transmissão.

O projeto tem como objetivo facilitar o acesso da comunidade às partidas transmitidas online, organizando eventos por data, campeonato e categoria.

## Funcionalidades

- Listagem de transmissões por data
- Organização por campeonato
- Exibição de informações do jogo (times, horário e plataforma)
- Contaúdo gerado pelo usuário

## Tecnologias

- Next.js (App Router / Server Actions)
- NextAuth 
- Drizzle ORM
- PostgreSQL
- Jest
- Biome (lint/format)
